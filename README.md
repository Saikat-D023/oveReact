# oveReact

A "quick-and-dirty" React clone built from scratch using raw TypeScript. I created this to learn how React works under the hood!

Currently, this is in its initial stage. It uses a "brute-force" rendering approach (wiping and recreating the DOM on every state change) and implements a basic `useState` hook. It is perfectly structured to serve as a foundation for building an advanced diffing algorithm and additional hooks like `useEffect` in the future.

## Setup & Running

This project uses Bun's built-in bundler to compile the TypeScript source code (`src/app.ts`) into a JavaScript file (`dist/bundle.js`) that can be run in the browser.

1. **Install Dependencies:**
```bash
bun install
```

2. **Start the Development Server:**
This will watch your `src/` files and automatically rebuild the `dist/bundle.js` bundle when you make changes.
```bash
bun run dev
```

3. **View the App:**
You can use any static server to serve the project. Using `live-server` is recommended for hot-reloading:
```bash
npx live-server
```

## Project Structure

- `index.html`: The main HTML file that hosts the application and loads the bundled script.
- `src/`: The core framework and application logic.
  - `app.ts`: Your application code where the `Counter` component lives.
  - `vdom.ts`: Virtual DOM creation (`createElement`) and DOM rendering (`renderNode`) logic.
  - `hooks.ts`: Implementation of global state memory and the `useState` hook.
  - `renderer.ts`: Component mounting logic and the `triggerRerender` loop.
  - `index.ts`: The framework's public API that exports all utilities.
- `dist/`: Contains the bundled output.
  - `bundle.js`: The browser-compatible JavaScript file generated by the `bun run dev` build process.
